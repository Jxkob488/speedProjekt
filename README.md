# Beach Party Gran Canaria

Statisches Event-Intro mit vier kurzen Intro-Seiten und einer finalen Event-Seite.

## UX-Verbesserungen
- Klare visuelle Hierarchie: Datum, Ort, Preis und Hauptaktionen sind schnell auffindbar.
- Fitts' Law: Große Buttons und gut erreichbare Navigation.
- Jakob's Law: Bekannte Muster wie Header-Navigation, Ticketkarten und CTA-Buttons.
- Hick's Law: Wenige, klare Auswahlmöglichkeiten statt überladener Bedienung.
- Kontrast und Lesbarkeit: Grelle Farben bleiben hinter gut lesbaren Textflächen.
- Barrierearmut: Fokuszustände, Skip-Link und `prefers-reduced-motion` für reduzierte Animation.
- Responsive Design: Layout passt sich an Smartphone- und Desktopgrößen an.

## Visuelle Effekte
- Neon-Farbwelt mit Pink, Cyan, Gelb, Lime, Orange und Violett.
- Animierte Muster, Partikel, Lichtflächen und geometrische Objekte.
- 3D-Hintergrund mit Three.js, sofern die CDN-Datei geladen werden kann.
- Sanfte Seitenübergänge ohne harte Strobe-Effekte.

## Start
Öffne `index.html` im Browser. Das Intro wechselt automatisch bis zur Event-Seite.

Direkt zur Hauptseite: `event-main.html`
